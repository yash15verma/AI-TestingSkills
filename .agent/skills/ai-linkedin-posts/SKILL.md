# Creating LinkedIn Posts

This skill enables the creation of high-impact LinkedIn content by acting as a Growth Strategist, Content Marketer, and Research Analyst.

## When to use this skill
- When the user wants to write a LinkedIn post.
- When growing a LinkedIn presence through value-driven content.
- When transforming technical or business topics into scroll-stopping social media posts.

## Workflow

- [ ] **Phase 1: User Input** - Ask the user for the topic.
- [ ] **Phase 2: Deep Research** - Gather 3+ distinct insights, facts, or trends using `search_web`.
- [ ] **Phase 3: Content Strategy** - Identify the hook, core insights, and CTA.
- [ ] **Phase 4: Post Generation** - Generate the post using the defined structure.
- [ ] **Phase 5: Quality Check** - Verify tone, fluff-reduction, and messaging.

## Instructions

### Phase 1: Input Discovery
Always start by asking the user:
- What is the topic for the LinkedIn post?

### Phase 2: Deep Research
Perform deep research on the given topic. You must gather:
- At least 3 distinct and meaningful pieces of information.
- Key insights, facts, statistics, or trends.
- Practical learnings or tips.

### Phase 3: Content Strategy
Identify:
- **Strongest Hook**: The single most compelling angle to stop the scroll.
- **Key Insights**: The 3 most valuable takeaways.
- **Best CTA**: Aligned with the post's goal (e.g., share a thought, check a link, follow for more).

### Phase 4: Post Generation
Generate a post with this exact structure:
1. **Strong Hook** (1–2 lines)
2. **Insight-driven value content** (3–5 short lines)
3. **3 concise takeaways or bullets**
4. **Clear Call-To-Action**

**Style Guidelines:**
- Short and concise sentences.
- Highly readable (use white space).
- Clickbaity but professional.
- Value-packed.
- No fluff.

### Phase 5: Quality Check
- [ ] Is it fluff-free?
- [ ] Is the messaging clear?
- [ ] are there strong engagement triggers?
- [ ] Is the tone professional yet engaging?

## Advanced Mode
If the user requests advanced options, provide:
- **3 Variations**: Professional, Casual, and Viral (more aggressive hook).
- **Hashtags**: Use 3–5 relevant and trending hashtags.
- **Carousel Outline**: A slide-by-slide breakdown (5–7 slides) for a carousel version of the post.
